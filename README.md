# Sihab Rahman — Portfolio

A minimalistic, responsive portfolio website showcasing my journey into **Data Analytics** and **Business Intelligence**, along with my design and development work.

## 🚀 Live Site

Deployed via GitHub Pages: [sihabrahman.github.io/sihabrahman](https://sihabrahman.github.io/sihabrahman)

## ✨ Features

- **Minimalistic design** — Clean layout, monochrome palette, modern typography
- **Expanding gallery** — Interactive project showcase with smooth CSS transitions
- **Responsive** — Fully mobile-friendly with adaptive navigation
- **Scroll animations** — Subtle fade-in and slide-up effects on scroll
- **No frameworks** — Built with pure HTML, CSS, and vanilla JavaScript

## 📂 Structure

```
/
├── index.html          # Main page
├── css/
│   └── style.css       # All styles
├── js/
│   └── main.js         # Navigation, animations, gallery
└── README.md
```
